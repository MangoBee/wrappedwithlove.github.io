# Wrapped With Love Florist Starter Site

A simple static website starter for a custom, request-based florist.

## Files

- `index.html`
- `style.css`
- `script.js`
- `/images`

## Image setup

Replace these placeholder image files with real bouquet photos:

- `images/hero-bouquet.jpg`
- `images/gallery-1.jpg`
- `images/gallery-2.jpg`
- `images/gallery-3.jpg`
- `images/gallery-4.jpg`

## Form behavior

The form currently creates a message, copies it to the customer's clipboard, and opens Instagram.

Later you can connect the form to Formspree, Netlify Forms, or Google Forms.
